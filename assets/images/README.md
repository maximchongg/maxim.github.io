# Images Folder

Place your images here:

## Required Images:

1. **profile.jpg** - Your profile photo (about section)
   - Recommended size: 500x500px
   - Square format works best

2. **project1.jpg** - Screenshot of your first project
   - Recommended size: 800x600px
   - Landscape orientation

3. **project2.jpg** - Screenshot of your second project
   - Recommended size: 800x600px
   - Landscape orientation

4. **project3.jpg** - Screenshot of your third project
   - Recommended size: 800x600px
   - Landscape orientation

## Tips:
- Use high-quality images
- Optimize images for web (use tools like TinyPNG or ImageOptim)
- Keep file sizes under 500KB for better loading times
- Use consistent dimensions for project images

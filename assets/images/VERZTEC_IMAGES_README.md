# Verztec Chatbot Project Images

Please add the following images to this folder:

## Required Images:

1. **verztec-team.jpg** - The award ceremony photo with the team and Verztec CEO
   - Use the photo showing the team holding the award certificate
   - Recommended size: 800x600px

2. **verztec-landing.png** - Screenshot of the chatbot landing page
   - Capture from: `public/index.html` or landing page
   - Shows "Intelligent Assistant for Modern Workplaces"
   - The dark themed landing page with "Get Started" and "Learn More" buttons

3. **verztec-chat.png** - Screenshot of the chat interface
   - Capture from: `public/chatbot.html` 
   - Shows the chat conversation with AI responses and document references
   - Should show the "What are the pantry rules?" example or similar

4. **verztec-analytics.png** - Screenshot of the analytics dashboard
   - Capture from: `public/analytics.html`
   - Shows the "Chatbot Analytics Dashboard" with usage stats
   - Topic distribution and recent chats

5. **verztec-avatar.png** - Screenshot of the 3D avatar
   - Shows the 3D avatar assistant (built with Blender/Mixamo)
   - The female avatar character in the chat interface
   - Can be from the avatar tab or chat interface

## How to Add:

1. Open your Verztec chatbot project
2. Run the application locally
3. Take screenshots of each page
4. Save them with the exact names above
5. Place them in this folder: `assets/images/`

## Alternative:
If you have existing screenshots, just rename them to match the names above and place them here.
